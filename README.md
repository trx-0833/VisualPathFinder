# VisualPathFinder
一个基于 Pygame 和 Tkinter 的交互式路径搜索算法可视化工具，支持BFS、DFS、Dijkstra、A*等多种经典搜索算法，并包含迷宫生成、图像导入和实时可视化功能。
An interactive pathfinding algorithm visualization tool based on Pygame and Tkinter, supporting multiple classic search algorithms such as BFS, DFS, Dijkstra, and A*, and featuring maze generation, image import, and real-time visualization capabilities.
